# TODO LIST

Simple python application to save a list of tasks and check them when done

## Table of Contents
- [Installation](#installation)
- [Features](#features)
- [Usage](#usage)
- [License](#licenses)

## Installation
1. Clone the repo
```bash
git clone https://github.com/
````

2. Run
```bash
python3 main.py
```

## Features
- Create a task
- Edit a task
- Mark a task as done or undone
- Delete a task

## Usage
- Enter a number to choose an action
1. Creates a task
2. Edits a task
3. Checks or unchecks a task
4. Deletes a task
